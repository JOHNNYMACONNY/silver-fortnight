## Challenge Types

1. **Daily Challenges** - Reset every 24 hours
2. **Weekly Challenges** - Reset every 7 days
3. **Milestone Challenges** - One-time achievements

## Challenge Tiers

1. **Bronze** - Entry-level challenges
2. **Silver** - Intermediate challenges
3. **Gold** - Advanced challenges
