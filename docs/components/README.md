## Component Documentation

### Trade Components
- **[CATEGORY_GRID.md](./CATEGORY_GRID.md)** - Visual category grid for browsing trade categories

### UI Components
- **[SLIDER.md](./SLIDER.md)** - Slider input component for numeric ranges and discrete values

## Component Categories

### Feature Components
Components specific to features (trades, collaborations, etc.):
- Located in `src/components/features/`
- Feature-specific functionality
- May depend on feature-specific hooks/services

### UI Components
Reusable, generic UI components:
- Located in `src/components/ui/`
- Framework-agnostic where possible
- Follow design system patterns
